TextAddon.com API

Welcome to the TextAddon.com API documentation.

Usage: https://textaddon.com/api/?bgUrl=Your image Url&t=Your text here&ff=font-face&c=color

e.g. https://textaddon.com/api/?bgUrl=https://yourwebsite.com/images/img1.jpg&t=TextAddon.com&ff=Flavors&c=red

  • You can choose the 'font-face' from the supported 'Font:' list in the Home page
  • You can choose the 'color' by its name or by the following hexadecimal code format (e.g. '0xff0000' for red).
    You can see a list of 140 color names and color codes here.
